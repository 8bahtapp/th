SketchUp ปัจจุบันเวอร์ชั่นอะไร▾

ปัจจุบัน SketchUp เวอร์ชั่น 2026

• Release Notes

ระยะเวลาการต่ออายุ▾

ต่ออายุล่วงหน้าได้ 60 วัน และล่าช้าไม่ควรเกิน 2 วัน
หลังจากครบกำหนดวันหมดอายุในระบบ จะใช้งานได้อีก 1-3 วัน

ความแตกต่าง SketchUp ในแต่ล่ะแผน▾

SketchUp Go - ใช้งานสำหรับ Web, ipad
SketchUp Pro - ใช้งานสำหรับ Web, ipad, Desktop
SketchUp Studio - ใช้งานสำหรับ Web, ipad, Desktop และมี V-Ray

• ตารางเปรียบเทียบ

Save as ไฟล์เป็นเวอร์ชั่นเก่าได้ไหม▾

SketchUp เวอร์ชั่น 2024 หรือใหม่กว่าไม่สามารถ Save as เป็นเวอร์ชั่นเก่าได้ โดยไฟล์จะสามารถเปิดและแก้ไขได้บน SketchUp 2021-2026

หากต้องการบันทึกไฟล์เป็นเวอร์ชั่นเก่ากว่าจะต้องติดตั้ง Extension เสริม
