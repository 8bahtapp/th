Chaos V-Ray เวอร์ชั่น 7 - ใบอนุญาตเช่าใช้

• Release Notes

ระยะเวลาการต่ออายุ▾

การต่ออายุจะเป็นแบบอัตโนมัติ ตามวันที่หมดอายุ การต่ออายุล่วงหน้าควรอิงจากระยะเวลาเสนอราคาและวันสิ้นสุดการยืนราคา
หลังจากครบกำหนดวันหมดอายุจะไม่สามารถใช้งานได้

Named license กับ Floating License ต่างกันยังไง▾

Chaos มีตัวเลือกใบอนุญาต 2 แบบ - ขึ้นอยู่กับแผนที่เลือก
• Named license - จำนวนผู้ใช้งานจะต้องตรงกับจำนวนใบอนุญาต (1 ใบอนุญาตต่อผู้ใช้งาน 1 คน)
• Floating License - แบ่งปันใบอนุญาตภายในองค์กรได้ ใช้งานได้พร้อมกันตามจำนวนใบอนุญาตที่มี
ในการสั่งซื้อใช้เพียงอีเมลเดียว สามารถจัดการใบอนุญาตได้ทั้งหมด

• รายละเอียดใบอนุญาตเพิ่มเติม
