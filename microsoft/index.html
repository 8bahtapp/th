FAQ

ปัจจุบัน Microsoft มีใบอนุญาตที่จำหน่ายกี่แบบ▾

ปัจจุบัน Microsoft มีใบอนุญาตที่จำหน่าย 3 แบบใหญ่ๆ
CSP: เหมาะสำหรับบริษัทมีหน้าจัดการใบอนุญาตชัดเจน แบบออนไลน์
ESD: เน้นใช้งานแบบส่วนบุคคลแบบออนไลน์ จัดการผู้ใช้งานไม่ได้
Box: เน้นใช้งานแบบส่วนบุคคลได้รับสินค้าเป็นกล่อง
เนื่องจากคู่ค้ามีการส่งสินค้าไม่เหมือนกัน ข้อมูลที่ได้รับอาจมีข้อมูลที่แตกต่างกัน

CSP คืออะไร▾

CSP (Cloud Solution Provider): รูปแบบการจำหน่ายสำหรับองค์กร จัดการใบอนุญาตออนไลน์ผ่าน Microsoft 365 admin center โดยต้องมีโดเมนเนมเช่น @domainname.onmicrosoft.com สำหรับสั่งซื้อ - มีทั้งเช่าใช้และซื้อขาด

ESD คืออะไร▾

ESD (Electronic Software Download): รูปแบบการจำหน่ายแบบออนไลน์ license จะถูกส่งไปที่อีเมลที่สั่งซื้อโดยตรง และผู้อยู่กับอีเมลนั้นไม่สามารถแอดผู้ใช้งานอื่นได้ - มีทั้งเช่าใช้และซื้อขาด

Box คืออะไร▾

รูปแบบการจำหน่ายแบบกล่อง ลูกค้าจะได้รับสินค้าเป็นกล่องซึ่งจะมี คีย์ สำหรับลงทะเบียน ลูกค้าสามารถนำ คีย์ นั้นไปลงทะเบียนไว้ที่อีเมลใดก็ได้ - มีทั้งเช่าใช้และซื้อขาด
